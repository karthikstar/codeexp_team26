# How to run the file

1. Download source
   ```bash 
   $ git clone https://github.com/karthikstar/codeexp_team26.git
   ```
3. Install Packages
   ```bash
    $ npm install
   ```
4. Test Run
   ```bash
    $ expo start
   ```
5. Open IOS Simulator / Web
    ```bash
      press i (for ioS Simulator) or w (for web)
    ```
